# Contributing

The smallest useful contribution is one of:

- a reproducible run proof;
- a contradiction with evidence;
- a failed case that the reference implementation does not handle;
- a privacy/security improvement;
- a translation with terminology review;
- an independent mirror heartbeat;
- a pull request that closes an accepted EvolutionProposal.

Do not submit private personal data, covert telemetry, claims of machine consciousness without an operational test, or code that automatically expands permissions.
